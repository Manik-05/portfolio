A responsive personal portfolio for **Manik Jindal**, an Information Technology student focused on AI/ML, computer vision, and full-stack development. It brings together selected projects, research, education, experience, certifications, and ways to get in touch.

**Live site:** [manik-05.github.io/portfolio](https://manik-05.github.io/portfolio/)

## Highlights

- Responsive single-page layout with accessible keyboard focus states and a mobile navigation menu
- Animated blueprint-style visual system, scroll progress indicator, active navigation states, and reduced-motion support
- Project cards covering AI/ML, computer vision, full-stack development, and browser applications
- Research, career timeline, certifications, and contact sections
- Built with plain HTML, CSS, and JavaScript - no framework or build step required

## Run locally

This is a static site, so opening it locally is enough.

```bash
git clone https://github.com/Manik-05/portfolio.git
cd portfolio
```

Open `index.html` in a browser. For live reloading during edits, run it through any local static server, for example:

```bash
npx serve .
```

## Customization

The site is intentionally self-contained. Update the following areas in `index.html` to tailor it:

- profile copy and social links
- featured project cards and their links
- research, timeline, and certification information
- the CSS variables in `:root` for colors, typefaces, spacing, and theme
